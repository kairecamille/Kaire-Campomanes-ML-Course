# Midterm

This folder contains my midterm audit project for ITAI 4374, focused on comparing ChatGPT to the human brain using neuroscience-inspired analysis and experimentation.

The project evaluates how artificial intelligence systems simulate cognitive functions such as:
- Memory and learning
- Emotional processing
- Selective attention
- Error correction
- Brain architecture
- Visual perception

Through a series of experiments and structured prompts, the report analyzes the strengths and limitations of AI systems compared to biological intelligence. The project also explores possible brain-inspired improvements for future AI development.

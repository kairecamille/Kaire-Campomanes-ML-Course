# Final Project

This folder contains the final project for ITAI 4374, focused on evaluating artificial intelligence systems through a neuroscience and cognitive computing perspective.

The project combines concepts from throughout the course, including:
- Brain-inspired AI architecture
- Cognitive system analysis
- Memory and learning models
- Attention and perception
- Emotional processing
- Human vs AI decision-making

The final project emphasizes critical analysis, experimentation, and technical research to explore how closely modern AI systems simulate human cognition and where significant differences still exist.

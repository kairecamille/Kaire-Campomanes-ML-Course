# Labs

This folder contains hands-on labs and neural simulation projects completed for ITAI 4374. The labs focus on applying brain-inspired AI concepts through experimentation, coding, and computational modeling.

Topics explored include:
- Leaky Integrate-and-Fire (LIF) neuron models
- Neural spike generation
- Membrane potential simulation
- Input current analysis
- f-I firing curves
- Noisy neural input behavior
- Brain-inspired computational systems

These labs helped develop a deeper understanding of how biological neuron behavior can be modeled mathematically and compared to artificial intelligence systems.

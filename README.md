# Kaire-Campomanes-ML-Course
This repository contains coursework, labs, and research projects completed for ITAI 4374, focused on the relationship between artificial intelligence and human cognition. Throughout the course, I explored how AI systems compare to the human brain in areas such as memory, perception, attention, emotional processing, and decision-making.

The projects in this repository combine:

Brain-inspired AI concepts
Neuroscience principles
Cognitive system analysis
Neural modeling simulations
AI behavior auditing
Scientific and technical writing

The course emphasized both theoretical understanding and hands-on experimentation using tools such as Wolfram Language, AI chat systems, and cognitive evaluation frameworks.

# Assignments

This folder contains written assignments and research projects completed for ITAI 4374. These assignments focus on comparing artificial intelligence systems with human cognition through neuroscience-inspired analysis, technical writing, and AI evaluation experiments.

Topics covered include:
- Brain architecture
- Memory and learning
- Emotional processing
- Attention and perception
- Human vs AI cognition
- Brain-inspired system design

The work in this folder combines scientific analysis, critical thinking, and hands-on experimentation with modern AI systems such as ChatGPT.
